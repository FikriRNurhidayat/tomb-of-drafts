# Snakowi
Makar
## How to run
Please have this following software on your machine:
* Ruby 2.6 or above
* Ruby 2D "Require Simple2D Library"
* OpenGL 2.0 or above "Simple 2D won't render anything if your OpenGL is below than 2.0"

Steps to run:
* ```bundle install```
* ```ruby app.rb```
